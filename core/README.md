Online Resume Builder using Django


Overview
The Online Resume Builder is a web application that allows users to create and generate professional resumes. Users fill out a form with their personal information, skills, education, and work experience. The app then generates a well-formatted resume in PDF format.

Features

 User-friendly form for inputting resume details
 PDF generation with proper formatting
 Secure user authentication
 Responsive design for various devices


Installation

1. Navigate to the project directory:

    cd core

2. Run database migrations:

    python manage.py migrate

3. Start the development server:

    python manage.py runserver

Access the app in your browser at http://localhost:8000/

Usage

Fill out the resume form with your details.
Click the “Download PDF” button.
Download your professionally formatted resume in PDF format.
